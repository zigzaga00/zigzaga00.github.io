    Tuples

    üü®üêç Technique Card: Python Tuples

    üß† What This Teaches

    How to create and use tuples in Python, how they're different from lists, and when you might want to use them.

    üß∞ You Will Use

    • () round brackets
    • Index numbers starting from 0
    • Tuple unpacking
    • .count() and .index() methods

    üß± What is a Tuple?

    A tuple is like a list ‚Äî it holds a group of items.
    But unlike a list, a tuple cannot be changed (we say it's immutable).

    fruit_tuple = ("apple", "banana", "cherry")
    • Items in a tuple are ordered
    • You can access them just like a list:
    print(fruit_tuple[1])  # banana

    üîí Why Use a Tuple Instead of a List?

    Think of a tuple like a sealed box. Once you put things in, no one can change what‚Äôs inside. This is helpful when:

    • ‚úÖ You want to protect your data so no one (even you!) changes it by accident.
    • ‚úÖ You‚Äôre sending or using fixed info, like coordinates, colours, settings, or labels.
    • ‚úÖ You want your code to run a little faster ‚Äì Python can deal with tuples more quickly than lists.
    • ‚úÖ You‚Äôre using dictionary keys ‚Äì only unchangeable things like strings and tuples can be used as keys.

    üß† Imagine:
    You‚Äôre storing a list of favourite numbers. But if that list should never change once chosen, using a tuple makes sure it stays safe and locked.

    favourites = (3, 7, 42)

    üß† Common Use Cases

    ‚úÖ Grouping coordinates:

    position = (10, 20)

    ‚úÖ Returning multiple values from a function:

    def get_info():
    return ("Alice", 10)

name, age = get_info()

    ‚úÖ Pairing data in a fixed structure:

    pair = ("dog", "bark")

    üö´ Tuples Can't Be Changed

    my_tuple = (1, 2, 3)
my_tuple[0] = 99  # ‚ùå This will give an error!

    üí° Tuple Tricks

    Tuple with one item:

    lonely = ("single",)  # Don't forget the comma!

    Use tuple() to convert a list:

    nums = [1, 2, 3]
nums_tuple = tuple(nums)

    üßÆ Methods You Can Use

    example = ("apple", "banana", "apple")

print(example.count("apple"))  # 2
print(example.index("banana")) # 1
